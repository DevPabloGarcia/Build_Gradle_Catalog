#### How to test it

Download this project

```sh
git clone git@github.com:DevPabloGarcia/Build_Gradle_Catalog.git
```

Go to project root in terminal and run

```sh
./gradlew versionCatalogUpdate
```
Check versions from the file ` libs.versions.toml`
